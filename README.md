# root-flowers-webapp
## Project Overview
This project was developed as part of the COS30020 Web Application Development course at Swinburne University of Technology Sarawak Campus.
The goal was to design a simple yet attractive website for Root Flowers, a local florist in Kuching, to showcase products, promote floral workshops, and support basic customer and administrative functions.

## Tech Stack
- **Frontend**: HTML, CSS, Bootstrap
- **Backend**: PHP
- **Data Storage**: Text files (.txt)
- **Tools**: XAMPP (Apache + PHP + MySQL)
  
## Features
- User account registration and login
- Profile management (including default profile image)
- Product catalog with categories and images
- Workshop information (venue, time, price)
- Student works showcase (to be extended in Assignment 2)
- About page with project details and demo video link

## How to Run
- Install and start XAMPP.
- Place the project files under htdocs/COS30020/.
- Open a browser and navigate to: http://localhost/COS30020/index.php
- Register a new account and explore the features.
